# MindMaster AI

## AI-Powered Learning & Memorization Coach

MindMaster AI helps you learn faster and remember longer using artificial intelligence.

Create personalized study plans, practice with AI-generated quizzes, review important concepts with Memory Pulse reminders, and track your learning progress over time.

---

## Features

- 🧠 AI-powered study plans
- 📚 Smart memorization workflows
- ❓ Interactive AI-generated quizzes
- 🔔 Memory Pulse spaced reminders
- 🔊 Text-to-Speech support
- 📈 Learning progress tracking
- ⭐ Premium learning experience
